Shell scripts for creating wiki tables for [1]:

mp-trac-(category|kde4|kde4-l10n)-ports.sh:
 - assumes that the MacPorts tree is installed on the system as an rsync'ed tree
 - create a tables of tickets for all ports of a certain category of parts of it


mp-trac-user-ports.sh:
 - assumes that the MacPorts tree is installed on the system as an SVN working copy
 - creates a table of tickets for all ports maintained by a certain user





[1] https://trac.macports.org/wiki/KDEProblems/KDETickets/KDE4-Category
